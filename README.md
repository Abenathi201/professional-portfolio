# Abenathi Sindapi - Portfolio

Personal portfolio website showcasing my work as a Junior Software Developer & QA Testing Specialist.

## Tech Stack

- **React 19** - UI library
- **Vite 7** - Build tool and dev server
- **TailwindCSS 4** - Styling
- **Framer Motion** - Animations and transitions
- **React Router DOM** - Client-side routing
- **TanStack Query** - Data fetching and caching
- **Lucide React** - Icon system
- **Sonner** - Toast notifications
- **html2pdf.js** - CV export functionality

## Features

- Animated loading screen with custom loader
- Responsive navigation and layout
- Project showcase with detailed views
- Interactive contact form
- Downloadable CV
- Custom UI components (glitch effects, grain overlay, halftone patterns)

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Project Structure

```
src/
├── Pages/          # Route pages (Home, Work, About, Contact, CV)
├── Components/     # Reusable components
│   ├── home/       # Home page specific components
│   ├── navigation/ # Navigation components
│   ├── footer/     # Footer component
│   └── ui/         # UI components and effects
└── main.jsx        # Application entry point
```
